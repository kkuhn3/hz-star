Core
These rules define the format.
Singleton
Making this a singleton format opens up folder options. Typical PVP folders contain about 3 copies of each chip. This means a 30 chip folder only contains about 10 unique chips, whereas this format will always have 30.
Chips of the same name but different codes are considered to be the same chip. For instance: VarSwrd C and VarSwrd D cannot be run in the same folder.
Star Chips () are banned
Star Chips are usually mega stop time navi chips. We removed these from the format in an attempt to diversify the folders. Without a blanket star chip ban, a lot of star chips were on the ban list such as BN2's Gospel, BN3's Flashman, and BN6's * coded navi chips. Removing star chips simplified the rules, while making matches feel different.
Bans
These options were deemed too strong for this format.
BN3's Folderback
BN3's Folderback is arguably the best chip across the whole series. Folderback works against the limitations of this format. Rather than chips being singleton, a player can reshuffle, and replay chips. This lessened the uniqueness of the format.
Convenience
These rules exist to make the format easier to participate in.
Online exclusives chips are banned
Requiring online in order to obtain a chip, or getting a chip through trade makes it harder to build the perfect folder. Instead this format can be played entirely on one save file, with no link cables needed.
Chip Trader exclusives chips are banned
Chip Traders are random, ignoring BN1's RNG manipulations. If a chip or code is only availabe via chip trader, it could require extensive time to obtain. So that folder building in this format is fun and straightforward, Chip Trader exclusive chips and codes are banned.
